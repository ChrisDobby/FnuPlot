FnuPlot Namespace

TypeDescription
Data

Data that are used as an argument to the 'Series' type For most of the types, we use 'Data', 'Function' is used when specifying function as a string
FillStyle

Represents possible fill styles of a plot
GnuPlot

Provides a wrapper for calling gnuplot from F#. Plots are drawn using the 'Plot' function and can be created using 'Series' type. For example:

// Plot a function specified as a string gp.Plot("sin(x)")

// Create a simple line plot gp.Plot(Series.Lines [2.0; 1.0; 2.0; 5.0])

// Create a histogram plot drawn using blue color gp.Plot(Series.Histogram(lineColor=Color.Blue, data=[2.0; 1.0; 2.0; 5.0]))
ICommand

Represents an abstract command that sets some property of the plot (and allows undoing the change)
Output
OutputType

Various outputs that can be specified to gnuplot
Series

Represents a series of data for the 'plot' function The type can be constructed directly (by setting 'plot' parameter to the desired series type) or indirectly using static members such as 'Series.Histogram'
Style

Represents a style of a plot (can be passed to the Plot method to set style for single plotting or to the Set method to set the style globally)
TimeFmtX

Used to specify datetime format for the x and y axes, if they contain time data. "format" is a gnuplot time format; see http://gnuplot.sourceforge.net/docs_4.2/node274.html
Titles

Used to specify titles for the x and y axes. For example:

// specify rotated titles for x axis Titles(x=["one"; "two"], xrotate=-70)
ModuleDescription
Internal
InternalFormat

Module that contains formatting of various gnuplot arguments
InternalUtils

Utilities for formatting values and for working with commands (this also contains workarounds for Mono bugs)
Ranges

Module with values for elegant construction of ranges
timeFormatting
